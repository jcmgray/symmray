[`symmray`](symmray) is a minimal library for block sparse, abelian symmetric
and fermionic arrays, designed to look as much as possible like standard
ndarrays, whose blocks can be backed by `numpy`, `torch` or any other
[`autoray`](https://github.com/jcmgray/autoray) compatible library.
